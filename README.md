# challenge
Back-end Challenge

## Requirements
- git
- Node v12
- npm v6
- express v4
- mocha v8

## Installation
To install the app locally:
```bash
git clone https://github.com/xosebastian/challenge.git
cd challenge
npm install
```
## Run
To run the app locally:
```bash
npm run dev
open http://localhost:5000
```

## Run Test
To run the app test:
```bash
npm run test
```
